## Troubleshooting

**"Please select a location first!" alert?**
- Click a **location** button before clicking a **model** button

**Models not appearing?**
- Check browser console (F12) for errors
- Models use remote URLs automatically (no setup needed)

**Performance issues?**
- Uncheck **3D Buildings**
- Set **Terrain** to 1.0x
- Click **Clear All Models**
