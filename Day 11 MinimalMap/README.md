## Design Principles
- **Simplicity** - Only essential features
- **Clean lines** - Minimal visual clutter
- **Balanced composition** - Thoughtful negative space
- **Monochrome palette** - Focus on form over color

## Workflow
1. Extract OSM data via Overpass Turbo
2. Import to QGIS for cleaning and processing
3. Export vector layers
4. Style in Canva for final minimalist design

## Datasets
- `Nukus_border.geojson` - City boundary polygon
- `Nukus_Roads.geojson` - Road network
- `Nukus_Waterway.geojson` - Water features

## Tools Used
- **Overpass Turbo** - OSM data extraction
- **QGIS** - Data cleaning and processing
- **Canva** - Final styling and composition

## Minimalist Cartography
This project emphasizes the "less is more" philosophy - removing unnecessary details to highlight the essential urban structure of Nukus.
