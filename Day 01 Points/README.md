# Day 01: Points

## Overview
Point data visualization for Tashkent showing art installations and public WiFi locations.

## Projects

### 1. Art Locations
![Art Locations](Art%20locations/artlocations.jpg)

**Data:** Art installation points across Tashkent
**Output:** Jupyter notebook analysis + GeoJSON export

### 2. WiFi Locations
![WiFi Locations](Wifi%20Locations/wifilocations.png)

**Data:** Public WiFi hotspot locations in Tashkent districts
**Output:** Interactive HTML map + CSV dataset
**File:** `Day_1_Points_Tumaris.html`

## Key Features
- Point-based spatial data collection
- District-level aggregation
- Interactive web visualization
- GeoJSON format for interoperability

## Tools Used
- Python (Jupyter)
- Leaflet.js (web mapping)
- GeoJSON
